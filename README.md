# matter-hub

Hub d'orchestration local (Westworld-style) pour :
- indexer l'historique ChatGPT (FTS + sémantique local)
- cataloguer les projets (détection, stacks, commandes)
- orchestrer des agents et automatisations (Clawdbot)

## Structure
- `registry/` : inventaire des projets + métadonnées
- `hub/` : scripts d'ingestion/indexation/recherche
- `dash/` : UI (plus tard)
